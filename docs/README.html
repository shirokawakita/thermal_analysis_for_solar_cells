GEO 太陽電池パネル
1年熱解析

静止軌道（GEO）3軸制御衛星の太陽電池パネルについて、1ノード・両面放射モデルで1年間の温度トレンドを算出します。

静的グラフ（PNG）に加え、SVG / Canvas アニメーションで β角・日食・効率感度を対話的に確認できます。概念図の元ネタ: Claude 共有解説

温度の求め方（概要）

パネルを1つの温度ノードとみなし、真空宇宙での 熱平衡（日射時） と 日食中の放射冷却 を組み合わせて年間トレンドを求めます。

黄道面・赤道面と季節

GEO 衛星は 赤道面（= 軌道面）上を周回します。地球の公転は 黄道面 上で行われ、地軸は 23.45° 傾いています。この傾きが季節と β角の年変動の根源です。

夏至≈ +23.45°なし全日照
冬至≈ −23.45°なし全日照

地軸の傾きと β角 Canvas

地軸傾き 0° なら β=0° が通年 → GEO では 毎日食 になります。

β角 — 南北面パネル・日食・年間推移

β角は太陽方向ベクトルと 軌道面（赤道面） のなす角です。GEO では太陽赤緯 δ と一致し、|β| < 8.7° で日食が可能です。パドル駆動の南北面パネルでは有効日射が S×cos(β) となります。

β角・南北面パネル・日食 — 側面図 + 年間チャート SVG アニメ

DOY
黄道面 赤道面 / 軌道 太陽電池パネル（N-S） β角 入射角 θ cos(β)
  • 春分・秋分（β ≈ 0°）… 太陽が赤道面内 → 日食シーズン
  • 夏至・冬至（|β| ≈ 23.45°）… 全日照、cos(β) ≈ 0.917
  • 有効日射 … $S_\mathrm{eff} = S \cos\beta$

日食（日陰）— 補足

上記インタラクティブ図で |β| < 8.7° のとき本影貫通（日食）が起きます。日食時間の年間分布は次のとおりです。

$$t_\mathrm{ecl}(d) = \max\bigl(t_\mathrm{spring}(d-81),\; t_\mathrm{autumn}(d-267)\bigr), \quad t_\mathrm{max}=72\;\mathrm{min}$$

南北面パネルと入射角 — 補足

パドル駆動の 南北面（N/S）パネル は季節の β を SADM で追従します。詳細は上記インタラクティブ図を参照してください。

  • 有効日射 … $S_\mathrm{eff} = S \cos\beta$（夏至・冬至でも cos(23.45°) ≈ 0.917）
  • 入射角 … パドル未補償分として $\theta \approx |\beta|$
デフォルトモデルは $S(d)$ をそのまま使用。β 補正版は感度解析のオプションです。

④ 日射平衡温度 Tsl

日食のない区間で、パネル前面・背面が宇宙へ放射する 両面放射 モデルとして定常熱平衡を解きます。

項式
日射吸収$\alpha_s (1-\eta_\mathrm{EOL}) S A_\mathrm{front}$
地球赤外$\varepsilon_\mathrm{back} \sigma T_\mathrm{earth}^4 F_E A_\mathrm{back}$
アルベド$\alpha_s S \rho_\mathrm{alb} F_E A_\mathrm{front}$

熱の出力（表裏で放射率を分離）

$$Q_\mathrm{rad} = \sigma \bigl(\varepsilon_\mathrm{front} A_\mathrm{front} + \varepsilon_\mathrm{back} A_\mathrm{back}\bigr) T^4$$

平衡温度（$A_\mathrm{front}=A_\mathrm{back}$、日射のみ）

$$\alpha_s (1-\eta_\mathrm{EOL}) S = (\varepsilon_\mathrm{front} + \varepsilon_\mathrm{back}) \sigma T^4$$

一般形:

$$T_\mathrm{sl} = \biggl(\frac{Q_\mathrm{in}}{\sigma\bigl(\varepsilon_\mathrm{front} A_\mathrm{front}+\varepsilon_\mathrm{back} A_\mathrm{back}\bigr)}\biggr)^{1/4}$$

上図は DOY 3（1月初旬）の例です。日射吸収が主な熱源であり、地球 IR・アルベドも加わったうえで、前面・背面からの放射と平衡します。本設定では 約 35〜40°C 程度の $T_\mathrm{sl}$ となります。

⑤ 日食最低温度 Tecl

$$m C_p \frac{dT}{dt} = -\sigma \bigl(\varepsilon_\mathrm{front} A_\mathrm{front} + \varepsilon_\mathrm{back} A_\mathrm{back}\bigr) T^4$$

本モデルでは最冷で約 −188°C 程度（秋の日食シーズン）です。

⑥ 軌道平均温度 Tavg

$$T_\mathrm{avg} = \bigl[(1-f) T_\mathrm{sl}^4 + f T_\mathrm{ecl}^4\bigr]^{1/4}$$

発電効率 ηEOL と温度プロファイル

$$Q_\mathrm{solar} = \alpha_s (1-\eta_\mathrm{EOL})\, S\, A_\mathrm{front}$$
